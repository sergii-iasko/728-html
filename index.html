There is a trio of Web Developement tools:

  1. HTML - for structure
  2. CSS - for styling
  3. JavaScript - for interaction

Each of them gives a specific functionality. But we are going to fovus on HTML tags which are just building block for the web page.

Main sematic tags:

  • head
  • body
  • header
  • main
  • footer

To add actual content we use certain tags which can help us to display content as we want it to do:

  • h1-h6 - to add a heading
  • p - to display a bunch of text
  • img - for displaying pictires
  • a - for the link to other website or to jump to another section of current page
  • input - different forms of fields where users can enter their data
  • div - to create a logical block of our content built with other tags
